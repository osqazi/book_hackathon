A comprehensive, AI-powered bilingual book platform built with Docusaurus, featuring advanced personalization, a RAG-based chatbot, and modern AI integrations. ## üöÄ Project Overview

This platform is a complete interactive book focused on **Physical AI & Humanoid Robotics**, built using the Docusaurus framework. It incorporates spec-driven development, AI-assisted coding, retrieval-augmented generation (RAG), user authentication, personalization, and full bilingual support (English & Urdu).

The entire stack was designed to demonstrate integration of cutting-edge AI technologies, multilingual UX, and personalized user experiences.

## üõ†Ô∏è Tech Stack & Key Features

- **Frontend & Documentation**: [Docusaurus](https://docusaurus.io/)  
- **Spec-Driven Development**: SDD-RI Spec-Kit Plus  
- **AI-Assisted Development**: Claude API (Claude Code) + Qwen API  
- **Intelligent Agents**: Custom Sub-Agents & reusable Skills with bilingual support  
- **RAG Chatbot**:  
  - Backend: FastAPI  
  - Vector Database: [Qdrant](https://qdrant.tech/)  
  - Embeddings: [Cohere](https://cohere.com/)  
  - Deployment: Hugging Face Spaces (Docker)  
- **Authentication & Personalization Backend**: Vercel  
- **Database**:  
  - Structured data: Neon PostgreSQL  
  - Vector store: Qdrant  
- **Bilingual Support**: English & Urdu with navbar language switcher  
- **User Personalization**:  
  - Onboarding questionnaire at signup  
  - Adaptive recommendations based on reading behavior
